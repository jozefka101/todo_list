# To-Do List Web App

This is a simple interactive To-Do List application built using HTML, CSS, and JavaScript. It allows users to add tasks, mark them as complete/incomplete, and remove tasks. The app saves your tasks in the browserâ€™s localStorage so that your list is preserved even after refreshing the page.

## Features

- Add new tasks with a text input box.
- Mark tasks as completed using checkboxes.
- Remove tasks using a trash can icon button.
- Persistent data storage with localStorage.
